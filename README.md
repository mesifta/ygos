# ygos

this project tries to build an operating system from scratch.

so far created
<ul>
    <li>a boot loader</li>
    <li>a basic 80x25 vga screen driver</li>
    <li>a keyboard driver</li>
    <li>a shutdown mechanism</li>
</ul>
    
next on the list is
<ul>
    <li>disk read/write driver</li>
    <li>a basic file system</li>
    <li> and much more..</li>
</ul>
